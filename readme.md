## Install

```
$ npm install --global alfred-scotts-awesome-wf
```

*Requires [Node.js](https://nodejs.org) 4+ and the Alfred [Powerpack](https://www.alfredapp.com/powerpack/).*


## Usage

In Alfred, type `swf`, <kbd>Enter</kbd>, and your query.
